# ADR-001: Product Name — cleo-rf

**Status:** Accepted
**Date:** 2026-04-08
**Decision Makers:** Product Owner + AI Architect

## Context

We need a working name for the Russian Cleo AI clone that is:
- Memorable and short
- References the original for clarity during development
- Has available domain/social media handles
- Doesn't violate trademark

## Options Considered

| Option | Pros | Cons |
|--------|------|------|
| **cleo-rf** | Clear reference to original, "-rf" = Russian Federation, easy to search | Too close to original trademark |
| **FinRoast** | Catchy, references killer feature | May limit product scope |
| **MoneyTalk** | Broad, friendly | Generic, hard to trademark |
| **Kopilka AI** | Russian word (piggy bank), AI reference | Old-fashioned connotation |
| **Trata** | Russian word for "spending" | Negative connotation |

## Decision

**cleo-rf** as internal/working name. Final consumer brand to be determined before launch through user testing. The "-rf" suffix clearly denotes Russian Federation adaptation.

## Consequences

- All internal docs, repos, and configs use "cleo-rf"
- Consumer-facing brand requires separate branding exercise before public launch
- No trademark conflict risk at development stage
